How to pick

Same copy, same link, same corner, same tokens — the only difference is visual loudness. A is theatrical, B is brand-heavy, C is quiet.

Cody's recommendation: B · Depth layers. It carries enough weight to be seen on the functional demos (plumbing, hvac, landscaping) without feeling pasted-on, and the layered-shade technique is the one LoungeLizard move that works at small component size. A needs space to breathe that a corner ribbon doesn't have; C disappears on busy hero sections.
